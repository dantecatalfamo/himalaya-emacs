* Himalaya Emacs
  Emacs front-end for the [[https://github.com/soywod/himalaya][himalaya]] email client.

  This is currently a work in progress.

  Most settings can be customized through the Emacs Easy Customize
  system. =M-x customize-group himalaya=

** List Messages

   =M-x himalaya= or =M-x himalaya-message-list=

   | Key   | Action           |
   |-------+------------------|
   | =n=   | Move cursor down |
   | =p=   | Move cursor up   |
   | =f=   | Forward page     |
   | =b=   | Backwards page   |
   | =j=   | Jump to page     |
   | =m=   | Switch mailbox   |
   | =w=   | Write message    |
   | =C=   | Copy message     |
   | =M=   | Move message     |
   | =RET= | View message     |

** Read Message

   After pressing enter on a message, you'll enter the message viewing
   mode.

   | Key | Action           |
   |-----+------------------|
   | =n= | Next message     |
   | =p= | Previous message |
   | =r= | Reply to message |
   | =R= | View raw message |
   | =q= | Kill buffer      |
